About

The Trend Analyser project is a Twitter data gathering and processing project that is being written as part of my 3rd year project at Southampton University.

Source Available

The source code to this application is available on github

Elements of the Trend Analyser

  • Hashtag Frequency Charts - This allows you to search for data in hashtags and visually represent how popular they have been over time.
  • Mention Frequency Charts - This allows you to search for a users Twitter handle and visually represent how they have been tweeted over time
  • Filter Terms - This allows you to see what current search terms are being processed and downloaded
  • Trend Search - This allows you to search the Twitter trending lists to see what and when terms have been trending
  • Current Twitter Trends - This allows you to view the most recent downloaded Twitter trends
  • Trend Locations - This allows you to view all the locations Twitter produces trending data for
  • Positive Wordlist - This lists all words deemed positive to be used by the semantic analysis
  • Negative Wordlist - This lists all words deemed negative to be used by the semantic analysis
  • Trending woeids downloading - This lists the locations that are currently having their official Twitter Trend list downloaded for
