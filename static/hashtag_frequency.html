Combined graph selector

  • Data that has been downloaded can be viewed by pressed the appropriate hashtag. This will be loaded fast since it is stored in memory, searching takes longer since it needs to search the database.
  • If you wish to view multiple hashtag data you can add elements to the right listbox by selecting them in the data already downloaded box and pressing add.
  • Remove will remove them from the combined graph selector box.
  • Pressing Generate will generate the graph with all the hashtag elements from the combined graph selector list.
